# منصة هداية (Hedaya Platform)

منصة إسلامية شاملة تجمع الأدوات التعبدية والذكر النافع في مكان واحد، مع دعم الذكاء الاصطناعي للإجابة على الأسئلة الإسلامية.

## المميزات

- 📖 **القرآن الكريم**: استماع للتلاوات بأصوات القراء المختلفين
- 📚 **الأحاديث**: مجموعة من الأحاديث النبوية الصحيحة
- 💰 **حاسبة الزكاة**: حساب الزكاة بدقة
- 🤖 **الذكاء الاصطناعي**: اسأل أسئلة إسلامية والحصول على إجابات من Grok AI
- 📅 **التذكير اليومي**: تذكيرات إسلامية يومية
- 🕌 **مواقيت الصلاة**: أوقات الصلاة حسب الموقع
- 🏛️ **المساجد القريبة**: العثور على أقرب المساجد
- 🌍 **الإسلام حول العالم**: معلومات عن المسلمين في دول العالم
- 📿 **المسبحة**: عداد التسبيح الرقمي

## المتطلبات

- Node.js (الإصدار 14 أو أحدث)
- npm أو yarn

## التثبيت والتشغيل

### 1. تحميل المشروع
```bash
git clone <repository-url>
cd hedaya
```

### 2. تشغيل الفرونت اند
```bash
cd frontend
# افتح index.html في المتصفح أو استخدم خادم محلي
# على سبيل المثال باستخدام Python:
python -m http.server 3000
```

### 3. تشغيل الباك اند
```bash
cd backend
npm install
npm start
```

الخادم سيعمل على: `http://localhost:5000`

### 4. إعداد مفتاح Grok API

1. احصل على مفتاح API من [https://console.x.ai](https://console.x.ai)
2. أضف المفتاح إلى ملف `backend/.env`:
```
GROK_API_KEY=your_api_key_here
```

## API الخلفية

### POST /api/ask-ai
يسأل الذكاء الاصطناعي أسئلة إسلامية.

**الطلب:**
```json
{
  "question": "ما هو أفضل الذكر؟"
}
```

**الرد:**
```json
{
  "answer": "أفضل الذكر هو لا إله إلا الله...",
  "question": "ما هو أفضل الذكر؟"
}
```

### GET /api/health
فحص حالة الخادم.

## التطوير

### تشغيل الفرونت اند في وضع التطوير
```bash
cd frontend
# استخدم خادم محلي مثل live-server أو python
python -m http.server 3000
```

### تشغيل الباك اند في وضع التطوير
```bash
cd backend
npm run dev  # يستخدم nodemon لإعادة التشغيل التلقائي
```

## البناء والنشر

### بناء الفرونت اند
الفرونت اند جاهز للاستخدام مباشرة. يمكن نسخه إلى أي خادم ويب.

### نشر الباك اند
```bash
cd backend
npm run build  # إذا كان هناك سكريبت بناء
```
